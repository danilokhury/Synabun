# Style Guide — Reference

Complete reference for Leonardo.ai styles, lighting, and visual options. Use this to make informed selections during the prompter questionnaires.

---

## Image Style Presets — When to Use Each

| Style | Best For | Description |
|-------|----------|-------------|
| Dynamic | General purpose, action | Balanced enhancement, good for most subjects |
| Cinematic | Moody, dramatic, film-like | Adds depth, dramatic lighting, film grain |
| Creative | Art, illustration, experimental | Painterly, artistic interpretation |
| Vibrant | Social media, marketing | Bold saturated colors, eye-catching |
| Portrait | Headshots, people, faces | Optimized skin tones, soft focus |
| Fashion | Editorial, high-fashion | Stylized lighting, fashion photography look |
| Stock Photo | Commercial, corporate | Clean, professional, stock photography |
| None | Full prompt control | No style modification — pure prompt |

**Guideline:** Use "None" when your prompt is already highly specific about style. Use "Dynamic" as a safe default. Use "Cinematic" for anything moody or dramatic.

## Video Style Stacking — Three-Layer System

Video styles stack in three layers. Pick one from each (or skip any). Only works with **Motion 2.0** and **Motion 2.0 Fast**.

### Vibe Layer (overall aesthetic)
| Vibe | Use When |
|------|----------|
| Pro Photo | Professional, polished look. Product videos, corporate |
| Stock Footage | Documentary feel, natural. Nature, travel |
| Sci-Fi | Futuristic, technological. Neon, holographic, cyber |
| Sketch | Hand-drawn, animated feel. Whimsical, explainer |
| Clay | Stop-motion, tactile. Playful, craft aesthetic |
| Color Sketch | Illustrated with color. Storyboard, artistic |
| Logo | Motion graphics, branding. Logo reveals, intros |
| Papercraft | Paper cutout, collage. Children's content, craft |
| None | Let other layers dominate |

### Lighting Layer (how light behaves)
| Lighting | Use When | Mood |
|----------|----------|------|
| Golden Hour | Warm, romantic, natural. Outdoors, sunset | Warm, nostalgic |
| Chiaroscuro | Dramatic contrast, cinematic | Intense, dramatic |
| Volumetric | Atmospheric fog, rays of light | Mysterious, ethereal |
| Lens Flare | Stylized, energetic, music video | Dynamic, glamorous |
| Backlight | Subject silhouette, halo effect | Dramatic, dreamy |
| Candle Lit | Intimate, warm, flickering | Romantic, cozy |
| Film Haze | Soft diffusion, vintage film look | Nostalgic, dreamy |
| Foggy | Atmosphere, depth, mystery | Eerie, contemplative |
| Hardlight | Sharp shadows, high contrast | Bold, edgy |
| Light Art | Neon, light painting effects | Creative, nightlife |
| Low Key | Dark, moody, minimal light | Suspenseful, noir |
| Luminous | Glowing, radiant, ethereal | Magical, uplifting |
| Mystical | Supernatural glow, fantasy | Enchanted, otherworldly |
| Rainy | Wet surfaces, reflections | Melancholic, moody |

## Motion Controls — Cinematic Language

### Camera Movement Controls
| Control | Cinematic Use | Emotional Effect |
|---------|--------------|-----------------|
| Dolly In | Push toward subject | Builds intimacy, focus |
| Dolly Out | Pull away from subject | Creates distance, reveals context |
| Dolly Left/Right | Lateral tracking | Following motion, exploration |
| Crane Up | Rising shot | Revelation, grandeur, power |
| Crane Down | Descending shot | Arrival, settling, intimacy |
| Crane Over Head | Overhead sweep | God's-eye view, scope |
| Tilt Up | Camera tilts upward | Awe, discovery, scale |
| Tilt Down | Camera tilts downward | Focus, descent, detail |
| Orbit Left/Right | Circles subject | Showcase, examination, drama |
| Rotate | Spins the frame | Disorientation, energy |
| Dutch Angle | Tilted frame | Unease, tension, dynamic |
| Snorricam | Fixed to subject's body | Intensity, vertigo |
| Robo Arm | Mechanical, precise moves | Technical, industrial |
| Lazy Susan | Slow rotation of scene | Dreamlike, display |

### Zoom Controls
| Control | Use | Effect |
|---------|-----|--------|
| Crash Zoom In | Sudden focus | Shock, emphasis, comedy |
| Crash Zoom Out | Sudden reveal | Surprise, context shift |
| Medium Zoom In | Moderate push | Gentle focus |
| Super Dolly In | Extreme forward push | Powerful drive, intensity |
| Super Dolly Out | Extreme pullback | Grand reveal |
| YoYo Zoom | In-out oscillation | Playful, hypnotic |
| Bullet Time | Frozen moment, rotating | Iconic, dramatic |
